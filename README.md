# MBJ_Mipmapping

A conversion to Swift of the example MBE_Mipmapping from the blog "Metal by Example" written by Warren Moore.

Converted to Swift by Jamnitzer (Jim Wrenholt)

Demostrates mipmapping on a simple cube.
